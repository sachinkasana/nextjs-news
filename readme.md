# News Next.js
A News feed app created with Next.js and Redux   
Using api from https://api.nytimes.com

## Running
### Development
```
git clone 
cd news-nextjs
npm install
npm run dev
```

### Production build
```
npm build
npm start
```

## Testing
### Unit Test
``` 
npm run test
```

### End-to-End Test
Make sure to start the local development as described above before running the test.
```
npm cypress
npm e2e // headless
```
